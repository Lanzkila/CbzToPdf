# 📚 Comic/Manga Converter v2
A simple, fast, and aesthetic web-based tool to convert your **CBZ** comic files into **PDF** or **EPUB** format directly in your browser.

![Screenshot](https://d378pye9mzk55i.cloudfront.net/Image20241212182458.jpg) 
*(Nota: Ganti link gambar ni dengan screenshot tool kau kalau nak)*

## 🚀 Features
* **No Server Upload**: Processing is done locally in your browser (privacy-focused).
* **Dual Format**: Supports conversion to both PDF and EPUB.
* **Custom Quality**: Adjust image quality and width to control file size.
* **Real-time Progress**: Visual progress bar with page counter (e.g., 1/24).
* **Manga Style UI**: Clean, modern interface with a subtle manga background.

## 🛠️ Tech Stack
* **HTML5 & CSS3**: Custom Bento/Cyberpunk inspired UI.
* **JavaScript (ES6)**: Core conversion logic.
* **JSZip**: For handling CBZ (ZIP) files.
* **jsPDF**: For PDF generation.

## 📖 How to Use
1.  **Select Format**: Choose whether you want PDF or EPUB.
2.  **Settings**: Set your desired image width and quality (default is recommended).
3.  **Upload**: Click **Choose File** and select your `.cbz` manga file.
4.  **Convert**: Hit **CONVERT NOW** and wait for the progress bar to finish.
5.  **Download**: Click the download link once processing is complete.
